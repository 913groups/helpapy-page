Un helper pour les développeurs pressés.

Simplifiez la gestion des dates, de la monnaie (CDF) et de la sécurité dans vos projets PHP.

$ composer require 913groups/helpapy

DATEFORMATTER

Formate une date en français avec gestion intelligente des styles.

// Exemple d'utilisation
use Groups913\Helpers\AppHelper;

echo AppHelper::DATEFORMATTER('2026-02-04', 'complet'); 
// Résultat : mercredi 4 février 2026

AGEFORMATTER

Calcule l'âge exact avec option de détails (ans, mois, jours).

echo AppHelper::AGEFORMATTER('1995-08-12', true); 
// Résultat : 30 ans, 5 mois et 23 jours

MONEYFORMATTER

Formatage monétaire pour la République Démocratique du Congo.

echo AppHelper::MONEYFORMATTER(15000.50); 
// Résultat : 15 000,50 FC

MASKFORMATTER

Protégez les données sensibles (Emails, Téléphones).

echo AppHelper::MASKFORMATTER('jean.dupont@gmail.com'); 
// Résultat : jea*******@gmail.com

VALIDJSONFORMAT

Validation rapide des flux JSON entrants.

if(AppHelper::VALIDJSONFORMAT($data)) {
    // Traitement sécurisé
}
